# cyclingchaos
A cycling data package built by CyclingChaos (CyclingChaos.co.uk) where you will be able to look at:
- Race Calendars
- Race Results
- Team Rosters
- Rider Details

Currently there is a four phase iteration delivery process:

- Phase 3 is adding data capability
